Offline Stadtplan auf MAPS-ME
Sie suchen eine bestimmte Adresse oder einen interessanten Punkt in München?! MAPS-Me bietet eine Navigationsfunktion, ist kostenlos und offline nutzbar.
So bekommen Sie den Stadtplan:

Downloaden Sie die App MAPS.ME. Sie finden sie im Android PlayStore oder Apple App Store. Sie ist kostenlos.
Downloaden Sie eine Karte Ihres Bundeslandes. München liegt in Bayern.
Downloaden Sie dann den Stadtplan für München

Download von MAPS.ME im  Android PlayStore oder im  Apple App Store
